My Projects

Animal Shelter Webpage

This is a webpage that I created to practice styling with CSS and separating areas of the page into columns (Before learning how to use Bootstrap. The webpage is intended to be for an animal shelter, displaying some of the animals that they have up for adoption. There are four different columns each displaying a different animal, and each column contains information about each animal as well as an image.

This project was created using HTML and CSS.

Island Resort Webpage

This is a webpage that I made to "advertise" a private island resort. The purpose of the project was for me to practice making a website containing multiple pages that would link to each other, as well as familiarizing myself with styling using CSS. The website starts at a homepage and you can then travel to five other pages to get more information about the resort. Each webpage also contains a link back to the homepage.

This project was created using HTML and CSS.

Vacation Webpage

This is a webpage that my partner Emily Dokken and I made together at Epicodus. Our task was to make a webpage about a fictional vacation that we went on and style it using Bootstrap. On our page we included blockquotes, background colors, helpers, columns and rows among other elements and classes.

This project was created using HTML and CSS.
